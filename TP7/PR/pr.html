Abstract

Criação de um servidor em nodejs utilizando a framework express. Este servidor vai responder operações CRUD. Irá ser utilizada uma base de dados de filmes e gerida a partir do mongoDB.

Esta aplicação irá mostrar os títulos dos filmes existentes na BD ,editar apagar, adicionar filmes e ainda vizualizar informações detalhadas de um filmes.

Ainda foram escritas algumas queries básicas para treinar a utilização do mongoDB.

Deliverables

  2. Funções js para executar o método PUT e editar um filme
