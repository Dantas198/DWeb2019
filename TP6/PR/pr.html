Abstract

Criação de um servidor em nodejs utilizando a framework express. Este servidor vai responder operações CRUD.

Esta aplicação irá mostrar as músicas existentes no ficheiro que serve como base de dados vai permitir editar, apagar, adicionar músicas e ainda vizualizar informações detalhadas de uma música.

Deliverables

  1. Funções js para executar o método DELETE e apagar uma música
  2. Funções js para executar o método PUT e editar uma música
  3. funções js para gerar id's no ficheiro json
  4. Stylesheet para os ficheiros html
  5. Rotas possíveis para toda a aplicação
  6. Métodos para ler e escrever no ficheiro json e executar a lógica dos métodos CRUD
  7. Template para o formulário de adicionar uma música
  8. Template para o formulário de editar uma música
  9. Template para a página de erro
  10. Template para a página principal da aplicação
  11. Template para o layout
  12. Template para a página de informações de uma música
  13. Ficheiro de configuração para a aplicação
  14. Ficheiro json que contém as músicas
