Abstract

Criação de um servidor em nodejs utilizando a framework express. Criação de um router para controlar pedidos à API e de routers para controlar os pedidos feitos à página web e que fazem pedidos à API. Irá ser utilizada uma base de dados de prémios Nobel e gerida a partir do mongoDB.

O métodos encontrados na API são os que foram pedidos no exercício 1 da ficha dweb2019-e2

Para além da API pedida foi criado um front-end para permitir a um utilizador fazer os pedidos que são permitidos. Este front-end é constituído por páginas de índice, premios, categorias e laureados, onde podem ser vizualizados os pedidos feitos à API.
