Abstract

Construção de um servidor em Node.js para responder a pedidos de páginas de arqueossítios

As páginas dos arqueossítios são obtidas partindo do ficheiro arq em diversas páginas xml de cada arqueossítio.

Estas páginas podem ser acedidas através de um indice. Ao aceder a uma página xml o servidor irá pedir uma stylesheet xsl para transformar a página xml em html.

Deliverables

  1. Stylesheet para transformar uma página arq.xml em arq.html
  2. Indice de arqueossítios por concelhos
  3. Stylesheet que constroi o index.html
  4. Stylesheet para separar os arqueossítios
  5. Servidor para atender pedidos
